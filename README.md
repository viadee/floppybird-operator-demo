# floppybird-operator

## prerequisite

- go version v1.19.0+
- docker version 17.03+.
- kubectl version v1.11.3+.
- Access to a Kubernetes v1.11.3+ cluster.

## project initialization

```shell
# create project
kubebuilder init --domain demo.viadee.de --repo github.com/viadee/floppybird-operator-demo
```
